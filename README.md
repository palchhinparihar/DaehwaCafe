# ☕ Daehwa Cafe

Daehwa Cafe is a community-driven platform dedicated to promoting Korean language learning, cultural exchange, and meaningful connections. It serves as a digital home where learners, enthusiasts, and community members can explore activities, services, leadership, and future initiatives.

## 🌟 Features

- Modern and responsive user interface
- Community-focused design
- About, Leadership, Services, Activities, and Future Projects sections
- Contact section with social links
- Smooth scrolling and subtle animations
- Optimized for desktop, tablet, and mobile devices

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- React Icons
- AOS (Animate On Scroll)

## 🚀 Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview the production build

```bash
npm run preview
```

## 🎨 Design Philosophy

The website emphasizes simplicity, accessibility, and a welcoming user experience. Soft gradients, clean layouts, and consistent visual hierarchy reflect the collaborative and cultural identity of the Daehwa Cafe community.
